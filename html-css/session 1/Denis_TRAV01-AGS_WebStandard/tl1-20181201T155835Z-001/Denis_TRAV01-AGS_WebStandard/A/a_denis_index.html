Créer des fonds de pages personnalisés

Il est possible de créer des fonds d'écrans particuliers. Assurez-vous que la photo est convertie en format .PNG puisque CSS ne suporte pas le format .JPG . L'image peut être soit répétitive ou simplement étendue sur toute la page avec la directive suivante:

                    
                        
                                                    body{
                                                        background-image:url(puce-pointe-a-droite.png);
                                                    }
